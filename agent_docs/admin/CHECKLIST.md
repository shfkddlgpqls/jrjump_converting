# ✅ Admin Implementation Checklist

## 1. 보안 및 권한 (Security)
- [ ] **로그인 인터셉터**: 비로그인 상태로 `/admin/**` 접근 시 로그인 페이지로 리다이렉트 되는가?
- [ ] **권한 분리**: '학원 관리자', '일반 관리자', '슈퍼 관리자' 등 레거시의 권한 레벨(`auth_level`)에 따른 메뉴 접근 제어가 동작하는가?
- [ ] **IP 차단**: 인가되지 않은 IP에서의 접근이 차단되는가? (설정된 경우)

## 2. 공통 기능 (Common)
- [ ] **레이아웃**: 상단 GNB, 좌측 LNB, 푸터가 모든 페이지에서 깨짐 없이 렌더링되는가?
- [ ] **파일 업로드**: 이미지/첨부파일 업로드 시 `application.yml`에 설정된 경로에 파일이 실제로 생성되는가?
- [ ] **에디터**: 게시판/상품 설명 등의 스마트 에디터(SmartEditor/CKEditor)가 정상 로딩되고 HTML이 저장되는가?
- [ ] **엑셀 다운로드**: 회원 목록 등에서 엑셀 다운로드 시 한글이 깨지지 않고 정상 출력되는가?

## 3. 도메인 연결 (Integration)
- [ ] **Academy**: 강사 목록 조회, 강사 프로필 수정이 정상 동작하는가?
- [ ] **Member**: 회원 검색 시 이름/아이디/전화번호 검색이 정확한가?
- [ ] **Banner**: 배너 등록 후 메인 페이지(Front)에 즉시 반영되는가? (캐시 확인)
- [ ] **Board**: 공지사항 등록 시 '상단 고정' 옵션이 정상 동작하는가?

## 4. 데이터 정합성 (Data Integrity)
- [ ] **목록 개수**: PHP 어드민의 목록 개수(Total Count)와 Java 어드민의 개수가 일치하는가?
- [ ] **상세 정보**: PHP 상세 페이지의 모든 필드 값이 Java 상세 페이지에서도 누락 없이 출력되는가?
