# ✅ Admin Test Guide

## 1. 테스트 전략

### 1.1. Security (권한)
- **ADMIN 권한**: `@WithMockUser(roles = "ADMIN")` 상태에서 정상 접근 확인.
- **권한 없음**: 비로그인 또는 일반 유저(`USER`) 접근 시 `403` 또는 `302`(로그인 리다이렉트) 발생 확인.

### 1.2. Controller (View)
- **Status**: `200 OK` 확인.
- **View Name**: `VIEW_MAP.md`와 일치하는 뷰 이름 리턴 확인.
- **Model**: 필수 데이터(목록, 페이징 정보)가 모델에 포함되었는지 확인.

## 2. 필수 체크리스트
- [ ] **로그인 인터셉터**: 비로그인 접근 차단 여부.
- [ ] **메뉴별 권한**: 특정 관리자 메뉴에 대한 접근 제어.
- [ ] **입력값 검증**: SQL Injection 및 XSS 방어 (검색어 등).
