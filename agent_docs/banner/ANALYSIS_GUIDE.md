# 🎨 Banner Analysis Guide (분석 가이드)

## 1. 분석 포인트
- **테이블 파편화**: `bannerTotalMain18`, `bannerTotalMain20` 등 연도별로 테이블이 나뉘어 있습니다. 소스 코드에서 `$year` 변수나 하드코딩된 테이블명을 찾아 패턴을 파악해야 합니다.
- **이미지 경로**: 배너 이미지의 저장 경로(`/data/File/...`)와 URL 매핑 규칙을 분석하십시오.

## 2. 핵심 정책 발굴 (Core Policy Discovery) - **Mandatory**
단순 코드 변환을 넘어, **기획자가 의도한 비즈니스 규칙(Business Rule)**을 찾아내야 합니다.

- **노출 정책**: "랜덤 노출인가, 아니면 광고주 계약에 따른 가중치(Weight) 기반 노출인가?"
- **기간 정책**: "종료일(`edate`)이 지난 배너는 즉시 사라지는가, 아니면 '종료' 배지로 남는가?"

> **Output**: 발견된 정책은 **반드시** `agent_docs/banner/POLICY.md` 파일에 기록해야 합니다.
