# ✅ Academy Implementation Checklist

## 1. 시즌 및 기초 정보
- [ ] **현재 시즌**: 메인 페이지 진입 시 현재 날짜에 맞는 시즌(`jr_season`)을 정확히 가져오는가?
- [ ] **과거 시즌**: 지난 시즌 데이터를 조회할 때 데이터가 올바르게 필터링되는가?

## 2. 강사 관리
- [ ] **목록 정렬**: PHP와 동일한 순서(`sort_no` ASC, 이름순 등)로 강사가 노출되는가?
- [ ] **이미지 경로**: 강사 프로필 이미지가 엑박 없이 출력되는가? (경로 매핑 확인)
- [ ] **정보 노출**: '사용 안함' 처리된 강사가 목록에서 제외되는가?

## 3. 캘린더/일정
- [ ] **월별 조회**: 달력 넘기기(이전달/다음달) 시 해당 월의 일정을 정확히 로딩하는가?
- [ ] **공휴일 처리**: 공휴일(`SETUP_HOLIDAY` 등)이 달력에 붉은색으로 표시되는가?

## 4. 레거시 호환성 (Legacy Compatibility)
- [ ] **인코딩**: 강사 소개글 등 장문 텍스트(`TEXT` 타입) 조회 시 한글 깨짐이 없는가?
- [ ] **하드코딩 로직**: 특정 강사 ID에만 적용되던 예외 로직(배너 숨김 등)이 Java에서도 동일하게 동작하는가?
