# ✅ Board Implementation Checklist

## 1. 동적 테이블 매핑
- [ ] **게시판 ID 연동**: URL의 `?bid=notice` 파라미터에 따라 `zetyx_board_notice` 테이블을 정확히 타겟팅하는가?
- [ ] **댓글 테이블**: 게시글 조회 시 해당 게시판의 댓글 테이블(`_comment`)에서 댓글을 가져오는가?

## 2. 게시글 기능
- [ ] **비밀글**: 비밀글 설정 시 리스트에서 '잠금 아이콘'이 표시되고, 내용 조회가 차단되는가?
- [ ] **공지사항**: '공지(Notice)' 속성이 있는 글이 리스트 최상단에 고정되는가?
- [ ] **조회수**: 글 읽기 시 조회수(`hit`)가 +1 증가하는가? (중복 방지 로직 포함 여부 확인)

## 3. 첨부파일
- [ ] **다운로드**: 첨부파일 클릭 시 다운로드가 정상 시작되며, 파일명이 깨지지 않는가?
- [ ] **이미지 본문 삽입**: 본문에 삽입된 이미지가 엑박 없이 출력되는가?

## 4. 권한/보안
- [ ] **쓰기 권한**: 읽기 전용 게시판에 글쓰기 버튼이 노출되지 않는지 확인.
- [ ] **수정/삭제**: 본인이 쓴 글(또는 관리자)만 수정/삭제 버튼이 보이는가?
