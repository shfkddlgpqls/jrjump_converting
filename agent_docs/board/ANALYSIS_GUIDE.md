# 📋 Board Analysis Guide

## 1. 분석 포인트
- **제로보드 구조**: `zetyx_` 또는 `bbs_` 로 시작하는 테이블 구조를 파악하십시오.
- **스킨(Skin)**: `bbs/skin/` 하위에 있는 스킨별 로직 차이를 확인해야 합니다. (Admin 설정에서 스킨을 선택함)

## 2. 핵심 정책 발굴 (Core Policy Discovery) - **Mandatory**
단순 코드 변환을 넘어, **기획자가 의도한 비즈니스 규칙(Business Rule)**을 찾아내야 합니다.

- **접근 제어 정책**: "비밀글은 작성자와 관리자만 볼 수 있는가? 답변 달린 비밀글은 원글 작성자도 볼 수 있는가?"
- **운영 정책**: "공지글(`notice`)은 항상 최상단에 고정되는가? 최대 몇 개까지 가능한가?"

> **Output**: 발견된 정책은 **반드시** `agent_docs/board/POLICY.md` 파일에 기록해야 합니다.

## 3. 체크리스트
- [ ] 게시판별로 테이블이 생성되는지(`zetyx_board_free`), 하나의 테이블에 `board_id` 컬럼이 있는지 확인했는가?
- [ ] 비밀글, 공지글(`notice`) 처리 로직을 분석했는가?
