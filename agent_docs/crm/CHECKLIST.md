# ✅ CRM Implementation Checklist

## 1. 메시지 발송
- [ ] **SMS 발송**: SMS 발송 시도 시 `CRM_SMS` 테이블에 정확한 데이터(수신번호, 내용, 발송시간)가 INSERT 되는가?
- [ ] **발신번호 검증**: 발신번호가 등록된 번호인지, 포맷이 맞는지 검증하는가?
- [ ] **메일 발송**: 메일 발송 시 HTML 형식이 깨지지 않고 전송되는가?

## 2. 상담 관리
- [ ] **상담 이력 저장**: 상담 내용 저장 시 줄바꿈, 특수문자가 보존되는가?
- [ ] **관리자 매핑**: 상담 등록 시 처리한 관리자(`manager_id`)가 정확히 기록되는가?

## 3. 쿠폰/포인트
- [ ] **쿠폰 발급**: 회원에게 쿠폰 발급 시 중복 발급 방지 로직이 동작하는가?
- [ ] **유효기간**: 발급된 쿠폰의 유효기간(`expire_date`)이 정확하게 설정되는가?
